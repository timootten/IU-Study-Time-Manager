# User Manual

This user manual explains how to use the IU Study Time Manager website from an end-user perspective.

## 0. Quick Start for New Users

1. Create an account or log in (see section 2).
2. Set language/theme (see section 1).
3. Create your first learning goal (see section 3).
4. Add a monthly plan for the current semester (see section 4).
5. Add your first session in the calendar or via timer (see section 5).
6. Enable notifications (see section 8).
7. Use the dashboard weekly for planned-vs-actual review (see section 7).

## 1. Introduction to the System

### Purpose
The website helps you plan, track, and evaluate study time in a structured way.

### Core Areas
- Dashboard: Overview of planned and completed study time, goal status, and key notices.
- Goals: Learning goals with period, target hours, milestones, and monthly planning.
- Calendar: Study sessions in month view, including manual entries and ICS import.
- Timer: Start/stop live sessions and add manual study time.
- Achievements: Manage outcomes (e.g., scores, grades, completed milestones).
- Settings: Profile, password, notifications, language, and theme.
- Admin (admin role only): User management, KPIs, and intervention options for user data.

### Navigation
- Top navigation: Home, About, optional Admin, plus language/appearance/account controls.
- Bottom navigation in dashboard areas: Dashboard, Goals, Calendar, Timer, Achievements.

### Language and Appearance
- Supported languages: German and English.
- Light/Dark mode via "Quick Settings".

## 2. Authentication and User Management

### Registration
1. Open the start page.
2. Select create account.
3. Enter name (min. 2 chars), email, and password (min. 8 chars).
4. Submit account creation.
5. After success, you are redirected to the dashboard.

### Login
1. Open login.
2. Enter email and password.
3. Confirm sign-in.

### Forgot Password
1. On the login page, select "Forgot password?".
2. Enter your email and request a reset link.
3. Open your email and use the link.
4. Set a new password and confirmation.

### Manage Profile
In profile/settings:
- Change name.
- Change email.
- Change password (current + new password).

### Logout
- Sign out from the profile menu in the header.

### Admin User Management (Admin only)
- The admin dashboard shows system KPIs.
- In Admin > Users:
  - Search, filter, and sort users.
  - Open user details.
  - Edit or delete users.
  - View/delete user goals and sessions when needed.

## 3. Learning Goal Planning

### Create a New Goal
1. Open Dashboard > Goals.
2. Create a new goal ("Create Goal").
3. Required fields:
   - Title
   - Category (e.g., Exam, Project, Presentation)
   - Time period (start/end)
   - Target hours
4. Optional: description, color.
5. Save.

### Filter and Search Goals
- Search field by title/description.
- Status filter (active, paused, completed, failed).
- Category filter (Exam, Project, Presentation, Other).

### Control Goal Status
In goal details:
- Change status directly (e.g., active -> completed).
- Check progress bar and remaining days.

### Edit/Delete Goal
- In goal details, use action buttons: edit (pencil) or delete (trash).

### Milestones
In goal details:
- Add milestone.
- Toggle completed/uncompleted.
- Edit/delete milestone.

## 4. Monthly Planning

### Create Monthly Plans
In goal details, in monthly plans:
1. Select add.
2. Enter month (YYYY-MM) and planned hours.
3. Optionally add notes.
4. Save.

### Update Monthly Plans
- Edit or delete existing monthly plans.

### Practical Use
- In the dashboard, check "Plan vs Actual" to detect over- or under-planning early.

## 5. Timer / Stopwatch

### Live Session
1. Open Dashboard > Timer.
2. Select a goal.
3. Press "Start Session" to run the timer.
4. Press "Stop Session" to store elapsed time.

### Add Manual Time
In timer area:
1. Select goal.
2. Set start time.
3. Enter duration in minutes.
4. Optionally add notes.
5. Save.

### Correct Time Entries
- Existing entries can be edited or deleted in goal details under "Sessions".

## 6. Goal Tracking

### Where Tracking Is Visible
- Dashboard area: total hours, plan progress, achievements.
- Goals area: status and progress per goal.
- Goal details: hour progress, milestone progress, sessions.

### Detailed Tracking in Goals
Goal details include three operational areas:
- Monthly Plans (rough plan)
- Sessions (detailed planning/time entries)
- Milestones

### Calendar-Based Tracking
In the calendar area:
- Monthly planning with day view.
- Planned/upcoming sessions.
- Color-coded sessions by category.

### ICS Import
In the calendar:
1. Open import.
2. Select ICS file.
3. Name and validate import.
4. Save import.

Additional options:
- Rename/update import.
- Toggle visibility of individual imports.
- Remove import including linked sessions.

## 7. Progress Visualization

### Dashboard Visualizations
- "Plan vs Actual" over multiple months.
- Goal progress as percentage + bar.
- Achievements by category.

### Achievements Area
- Record outcomes (e.g., grades, scores, notes, goal link).
- Filter by category/goal and search terms.
- Open details, edit, and delete entries.

### Interpretation Recommendation
- Review "Plan vs Actual" regularly (e.g., weekly).
- Combine low goal completion with milestone checks.
- Use achievements for qualitative reflection besides pure hour metrics.

## 8. Additional Website Features

### Legal
- "Imprint", "Privacy", and "Terms" are available via the footer.

### PWA Features
- Installable as app ("Install app", if browser supports installation).
- Service Worker for offline support.
- Offline view when no connection is available.

### Cookie Notice
- Informs about technically necessary storage only.
- Can be confirmed and dismissed.

### Notifications
In settings:
- Enable/disable email reminders.
- Define lead time before sessions.
- Enable/disable push notifications.
- Set "Quiet Hours".
